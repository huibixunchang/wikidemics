mysql> CREATE DATABASE wikistats DEFAULT CHARACTER SET utf8 COLLATE utf8_unicode_ci;
mysql> CREATE USER 'wikistats'@'localhost' IDENTIFIED BY 'password';
mysql> GRANT ALL ON wikistats.* TO 'wikistats'@'localhost' IDENTIFIED BY 'password';

JDBC URL:
    jdbc:mysql://localhost:3306/wikistats?user=wikistats&password=passwd

Test JDBC Connector by running the jdbc.java file:
    java jdbc

To run the spark script, the class path for the JDBC connection must be provided:

    /usr/local/spark/bin/spark-submit --jars "/home/william/Downloads/mysql-connector-java-5.1.40/mysql-connector-java-5.1.40-bin.jar" ~/wikidemics/wiki_agg.py
